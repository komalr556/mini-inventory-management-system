# Mini Inventory Management System

A Python-based CLI application to manage suppliers, products, stock, and discounts using SQLite database.

## Features
- Add and manage suppliers and products
- Update stock quantities
- Search products by name
- Low stock alerts
- Discount management
- Export inventory report to file
- Persistent storage using SQLite

## Technologies Used
- Python
- SQLite
- SQL
- File Handling

## How to Run
python main.py
